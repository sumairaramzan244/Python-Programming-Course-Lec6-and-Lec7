## What I Learned
* `str`, `int`, `float`, and `bool` are the four basic data types in Python
* `Tuple` is an ordered and unchangeable collection written in `( )`
* `List` is an ordered and changeable collection written in `[ ]`
* `Dict` is a collection of key-value pairs written in `{ }`
* `+` operator adds numbers and joins strings together
* `-` operator subtracts one number from another
* `*` operator multiplies two numbers
* `/` operator divides and always returns a float
* `//` operator divides and returns only the whole number part
* `%` operator returns the remainder after division
* `**` operator is used for power or exponent calculation
* We can mix comma ( , ) and plus ( + ) in print to display variables and strings together
